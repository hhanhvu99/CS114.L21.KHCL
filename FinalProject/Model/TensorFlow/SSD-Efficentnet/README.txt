Thư mục này chứa model EfficientDet, tất cả các thư mục dưới đều nằm trong Drive của nhóm:
best_model: checkpoint của model tốt nhất trong EfficientDet
custom_model: model của nhóm, trong đó chứa file config và các checkpoint
pretrained_model: model có sẵn trên TensorFlow, chứa pretrained model
saved_model: nơi export model dùng để chạy demo
